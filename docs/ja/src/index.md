# Genie

## 生産性の高いJulia Webフレームワーク

Genieは、モダンなWebアプリケーションを開発するための合理化された効率的なワークフローを提供するフルスタックMVC Webフレームワークです。 Juliaの長所（高レベル、高性能、動的、JITコンパイル）に基づいて構築され、生産的なWeb開発のための豊富なAPIと強力なツールセットを公開します。

### 現在の状態

GenieはJulia v1.0以降と互換性があります。
